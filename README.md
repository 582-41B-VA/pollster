# Pollster

Pollster is a website where people can create and respond to polls. It
is built on [Django]. To run the project locally, start by cloning this
repository on your computer. Then, run `uv run manage.py migrate` to
create the database and apply migrations. The
`uv run manage.py runserver` command is used to start the development
server.

[Django]: https://www.djangoproject.com/
